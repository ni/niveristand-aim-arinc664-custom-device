Tx Generic

This section implements a Generic Transmission Operation in the specific port instantiated. The table below shows the frames loaded from the Configuration xml file and their properties. Refer to AIM documentation for details.

This section has the following grouping:

  • Payload —A section for each specific payload. Multiple frames could use the same payload.
  • Tx Generic Commands —This section includes commands to control raw frame generation.
  • Tx Generic Status —VeriStand Channels that are used to see the "health" of the Tx Generic mode.
