Monitor Utilization

Define different VeriStand Channels with Statistics of the received data and the current busload.

This section has the following VeriStand Channels representing the following statistiscs:

  • TotalByteCount—Counter of the total bytes received since the last counter reset.
  • FrameGoodCount—Counter of the error-free frames received since the last counter reset.
  • FrameErrorCount;—Counter of all erroneous frames received since the last counter reset.
  • BytesPerSecond;—Instantaneous bytes received per second. This value is updated at a fixed interval.
  • FramesPerSecond;—Instantaneous frames received per second. This value is updated at a fixed interval.

    • This section has the following configuration settings:

    • Name—Specifies the name of the section in the system definition.
    • Description—Describes the section.
