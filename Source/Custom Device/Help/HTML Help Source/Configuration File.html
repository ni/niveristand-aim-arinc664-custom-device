Configuration File

A parameter configuration file is required to run the AIM ARINC 664 custom device. The following table describes this file.

The parameter configuration file configures the custom device to encode and decode parameters. The file is required for AIM ARINC 664 system definitions created in System Explorer.

For more information on the parameter configuration file schema, refer to the AIM ARINC 664 Custom Device GitHub repository.

This section has the following configuration settings:

  • Parameters—Specifies the path to the parameters configuration file.
  • Refresh—Invokes the Refresh dialog box. Use this dialog box to view changes to a configuration when importing a new parameters configuration file or when the parameters configuration file has changed on disk.
