AIM ARINC 664

The AIM ARINC 664 Custom Device configures AIM GmbH hardware designed for avionics communications using the ARINC 664 protocol. This addon configures only one AIM device. You may use multiple AIM ARINC 664 addons in your VeriStand project.

What do you want to do?

Depending on your goal, complete the following task.

Goal Task
View/Edit Hardware Configuration Configure the hardware device on this page.
Import Parameter Configuration Import the parameter configuration file in the Configuration Files section.
View Hardware Ports, and Parameter Configurations Select a hardware Ports, or a named parameter in the system tree.

Device Configuration

This device has the following configuration settings:

  • Name—Specifies the custom device name.
  • Description—Documents this instance of the custom device.
  • Initialization Method—Specifies the method used to assign the board to this custom device:
    • Board Number—Assign the board by its identification (ID) number. It starts at 0.
    • Serial Number—Assign the board by its serial number.
  • Description—Documents this instance of the custom device.
