Monitor Error

Define different VeriStand Channels with Statistics about Errors in frames.

This section has the following VeriStand Channels representing the following error statistiscs:

  • PhysErrorCount—PHY: Wrong physical symbol during frame reception.
  • PreamErrorCount—Wrong Preamble/Start Frame Delimiter received.
  • UnaligErrorCount—Unaligned frame length received (Triple Nibble Error).
  • CRCErrorCount—MAC CRC Error.
  • IFGErrorCount—Short Interframe Gap Error (<960ns).
  • IPErrorCount—AFDX IP Framing Error (AFDX-IP frame specific settings violated).
  • MACErrorCount—AFDX MAC Framing Error (AFDX-MAC frame specific settings violated)
  • NoSfdErrorCount—Frame without valid Start Frame Delimiter received.
  • VLLenErrorCount—VL specific Frame size Violation
  • SNIntegrityErrorCount—Number of frames detected with sequence number integrity error.
  • TrafShapingViolationErrorCount—Traffic Shaping Violation.

    • This section has the following configuration settings:

    • Name—Specifies the name of the section in the system definition.
    • Description—Describes the section.
